# multiple-pages

> Multiple pages starter project.

## Dependencies

* Zepto
* Webpack
* Less
* Babel

## Usage

Make sure installed `workplus-cli`.

```bash
$ npm install workplus-cli -g
```

And run:

```bash
$ workplus start multiple-pages my-project
```

## Build Setup

```bash
# install dependencies
npm install

# dev
npm run run

# run production build
npm run build

```
